🌍 Air Quality Analytics and AQI Prediction
📌 Project Overview

Air pollution has become one of the most significant environmental and public health concerns worldwide. This project analyzes historical Air Quality Index (AQI) data to identify pollution trends, compare air quality across cities and seasons, and develop machine learning models capable of predicting AQI.

The project combines Python, PostgreSQL, SQL, Machine Learning, and Power BI to transform raw environmental data into actionable insights.

🎯 Objectives
Clean and preprocess raw AQI data.
Store and manage data using PostgreSQL.
Perform SQL-based business analysis.
Analyze pollution trends using Python.
Build machine learning models to predict AQI.
Develop an interactive Power BI dashboard.

📌 Key Insights
PM2.5 is the strongest contributor to AQI.
Winter season experiences the highest pollution levels.
Certain cities consistently report poor air quality.
Random Forest achieved the best predictive performance.
Machine learning effectively predicts AQI using pollutant concentrations.
